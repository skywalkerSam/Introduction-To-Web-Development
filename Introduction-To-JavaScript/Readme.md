## Calculations with Numbers & Strings

### 3 + "6"
- `"36"`

### 9 - "6" 
- `3`

### 9 % 3
- `0`

### "Java" + "Script"
- `"JavaScript"`

### " " + " "
- `''`

### " " + 0 
- `'0'`

### true + true
- `2`

### true + false 
- `1`

### false + true 
- `1`

### true - false
- `1`

### false - true
- `-1`

### 6 - 9 
- `-3`

### "Starboy" - "Sam" 
- `NaN`

### 9 >= 6 
- `true`

### 0 === 1 
- `false`

### 9 <= 6 
- `false`

### 0 !== 0 
- `false`

### "A" > "B" 
- `false`

### "B" < "C" 
- `true`

### "a" > "A" 
- `true`

### "b" < "A" 
- `false`

### true === false 
- `false`

### true != true 
- `false`
