# Backend server for [face-detection](https://github.com/skywalkerSam/face-detection)

- Express.js
- bcrypt
- postgreSQL
- Knex.js


## Using `nodemon`, put this under "scripts" in package.json

```json
"start": "nodemon server.js"
```

## Install nodemon
```shell
npm i nodemon
```

## Start the server
```shell
npm start
```
